SwiftLink Telecom Management 🚀
A dynamic, full-stack web application designed for seamless telecom management, built to solve data synchronization and UI mismatch issues in modern service portals.

🛠️ Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+)

Backend: Python (Flask)

Database: SQLite (Automated deployment)

✨ Key Features
Real-Time Session Synchronization: The frontend perfectly mirrors the backend session state to prevent profile mismatches.

Intelligent Offer Triage: A dynamic, randomized promotional rewards system that locks upon claiming.

Strict API Handshakes: JSON-enforced client-server communication to prevent data parsing errors (like HTTP 415).

Regex Security: Client-side form validation ensuring strict 16-digit card limits and proper UPI formats.

Automated Database Generation: Backend scripts automatically deploy and upgrade SQLite tables upon server startup.
